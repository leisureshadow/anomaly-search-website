## How to Use

### Local Development

1. Simply open `index.html` in any modern web browser
2. No build process or server required

### Deployment Options

**GitHub Pages:**
1. Create a GitHub repository
2. Upload all files to the repository
3. Go to Settings â†’ Pages
4. Select the branch and root folder
5. Your site will be live at `https://yourusername.github.io/repository-name`

**Netlify:**
1. Drag and drop the entire folder to Netlify
2. Your site will be live instantly with a free domain

**Vercel:**
1. Import the folder to Vercel
2. Deploy with one click

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    --accent-color: #3b82f6;
    /* ... */
}
```

### Content
All content is in `index.html`. Each section is clearly marked with comments.

### Language Toggle
The language toggle automatically switches between English and Chinese content using the `.en-content` and `.zh-content` classes.
